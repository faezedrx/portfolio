## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/faezedrx/portfolio.git
   cd portfolio
   ```
   
2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```
   
3. Run the Streamlit app:

   ```bash
   streamlit run portfolio.py
   ```
